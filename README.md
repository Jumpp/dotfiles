# dotfiles

There are many like it, but these are mine

## What's in here

- **Sway** - tiling window manager
- **Waybar** - status bar
- **Rofi** - application launcher
- **Foot** - terminal
- **Neovim** - text editor
- **Mako** - notifications
- **Swaylock** - screen locker
- **Kanshi** - monitor management
- Plus some scripts etc

## Setup

Clone this repo and run the link script:

```bash
git clone git@github.com/Jumpp/dotfiles-basic.git ~/dotfiles
cd ~/dotfiles
./link.sh
```
